Project: Weather Dashboard
Description-
A dynamic Weather Dashboard application that provides real-time weather information for any city. This project demonstrates the power of asynchronous JavaScript to fetch and process data from a public REST API.

Technologies Used-
HTML5: Structured the semantic web layout.
CSS3: Designed a responsive and modern user interface.
JavaScript (ES6+): Implemented core logic, including:
Fetch API: To make asynchronous HTTP requests to the weather service.
Async/Await: For clean and efficient handling of asynchronous operations.
JSON Parsing: To extract and display complex data structures.
DOM Manipulation: For dynamic UI updates based on user input.
REST API: Integrated a third-party weather API (OpenWeatherMap API) to retrieve live data.

Key Features-
.Real-time search functionality for any city.
.Displays live weather metrics: Temperature, Humidity, and Wind Speed.
.Error handling for invalid city names or network issues.
.Fully responsive design suitable for mobile and desktop screens.
